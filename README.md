# Jerbawise

Jerbawise is a desktop tool developed in Python using the PyQt6 graphical library. It allows users to download musical content and manage a local library in a simple and visual way.

## Installation
If you just want to use the software, you don't need to download the source code:
1. Go to the **[Releases]** section (on the right side of the page).
2. Download the `Setup_Jerbawise_v1.exe` file.
3. Install it on your computer and launch it directly from your Desktop.

## Built With
* Python 3
* PyQt6 (Graphical User Interface)
* Inno Setup (Packaging and deployment)

## Disclaimer
This project has been created **strictly for educational purposes** and to serve as a technical demonstration of library integration in Python.
